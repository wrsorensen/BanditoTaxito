# Bandito Taxito Assets

Use this folder for branding/image assets only.

## Folder guide

- `logo/` — app logo, badge logo, wordmark
- `mascot/` — three-legged brindle dog mascot, bandit/dog hero art
- `icons/` — receipt, mileage, tax stash, work log, reports, 1099 icons
- `backgrounds/` — parchment, desert night, texture images
- `app-icon/` — app launcher icons such as 192x192 and 512x512
- `mockups/` — screenshots and UI mockups
- `source-drafts/` — rough drafts, large originals, alternate image generations
- `approved-app-ready/` — compressed assets approved for the live GAS app

## Rules

- Do not upload private receipts, tax records, Sheet IDs, API keys, or client-sensitive data.
- Keep large drafts in `source-drafts/`.
- Use compressed `.png`, `.webp`, or `.svg` files for live app assets.
- Only use `approved-app-ready/` for images that are safe to reference from the app.
