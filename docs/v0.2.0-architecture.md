# Bandito Taxito v0.2.0 Architecture

## Current target setup

GitHub Pages is the full branded frontend.

Google Apps Script is the backend/API only.

Google Sheets and Google Drive remain the storage layer.

## Root files

Required in GitHub root:

- assets/
- docs/
- releases/
- Code.gs
- index.html
- appsscript.json
- README.txt or README.md
- .nojekyll if GitHub accepts/keeps it

Do not use capital-I `Index.html` for the v0.2.0 GitHub-fronted architecture.

## Apps Script

Apps Script only needs:

- Code.gs

Old Apps Script `Index.html` can remain in the GAS editor, but it is unused by v0.2.0.
